Información general de la aplicación

Esta aplicación permite calcular el potencial de generación de energía renovable, tanto solar fotovoltaica como eólica, en una ubicación específica. Para obtener resultados más precisos, se recomienda usar la API externa PVGIS, que proporciona datos reales y detallados. Sin embargo, para quienes prefieran simplicidad, ofrecemos nuestra propia API interna para cálculos de energía fotovoltaica y eólica.

Secciones principales

Calcular con PVGIS: Utiliza la API externa para obtener datos precisos y confiables del potencial solar.

Calcular Fotovoltaico: Realiza cálculos y muestra resultados basados en nuestra API interna para energía solar.

Calcular Eólica: Cálculos y visualización del potencial eólico usando nuestra API propia.

Recuerda que es necesario estar registrado e iniciar sesión para acceder a estas funcionalidades.
